# HisQu Landing Page

Landing Page für das Projekt (HisQu) mit Fokus auf Storytelling, Projektbezug und einer anschaulichen Darstellung des Forschungsprozesses.

## Überblick

Diese Anwendung ist eine statische Website mit Storytelling-Design.  
Sie wurde entwickelt, um

- den Forschungsgegenstand `RG` verständlich zu machen,
- den iterativen Arbeitszyklus des Projekts visuell aufzubereiten,
- Projektpartner, Team und Förderer (DFG) klar und transparent darzustellen.

Die aktuelle Version wurde am 30.10.2025 an HisQu übergeben.

## Features

- Storytelling-basierter Seitenaufbau (Hero + Prozessrad + inhaltliche Sektionen)
- Visualisierung des Forschungsprozesses als Prozessrad mit Ankern zu Einzelsektionen
- Mehr Projektbezug im Hero (Bezug zum RG und zu den geschichtlichen Erkenntnissen)
- Zweisprachige statische Inhalte (Deutsch/Englisch) \[Optional/Geplant\]
- Bereich \`Meet the partners\` mit nachgelagertem \`Meet the team\`
- Einbindung von Beispielseiten aus dem RG (z\.B\. als Faksimile-Ausschnitt)
- Darstellung der Förderinformationen (DFG, Fördernummer) im Footer bzw\. unter dem Hero
